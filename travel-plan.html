Travel Plan

About the Project

My goal for this project was to build a Drupal 7 site that would allow users to create itineraries and maps for upcoming trips. In order to create this functionality, I built a custom module that:

  • Allows the user to enter a city and type of attraction to search for
  • Uses the Google Geolocation API to determine the latitude and longitude of the city
  • Uses the city's latitude and longitude, along with the attraction type to return attractions using the Google Places API
  • Shows the user attraction results, allowing them to add specific attractions to their itinerary; each added item is saved as a node
  • Tracks trips for multiple different cities for each user
In addition to my custom module, I also used the IP Geolocation Views & Maps module to display a map of attractions for each trip.
